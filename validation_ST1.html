Reflection on Validation

During validation, I encountered several heading hierarchy errors across the pages. The Home Page initially had an <h4> following an <h2>, skipping the <h3> level. I corrected this by changing all gallery tile headings from <h4> to <h3>. Similarly, the Content Page had an <h3> immediately following the <h1> page title, which I fixed by changing it to <h2>.

The Gallery Page also had a section lacking a heading and an image with an empty src attribute. I resolved these by adding a hidden heading to the section and providing a placeholder image source.

This validation process reinforced the importance of maintaining proper semantic heading hierarchy and ensuring all elements have valid, non-empty attributes, as these are fundamental to both accessibility and standards compliance.
