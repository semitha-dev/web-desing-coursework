Reflection on Validation

I initially had a validation error because the meta refresh tag was missing the content attribute value separator. Correcting the format to content="4;url=home.html" resolved this. I also learned that embedded style blocks must be placed within the <head> for valid HTML5, which I had initially placed incorrectly in the body.
